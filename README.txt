"# txt_sentoken is the dataset that we are using to train and test the sentiment of tweet. It contains positive and negative tweet in each separate folder.
The link to download dataset http://www.cs.cornell.edu/people/pabo/movie-review-data/
Place folder txt_sentoken in C Drive. 
This algorithm first trains the machine on half data of negative sentiment and half data of postive sentiment using Naive Bayes Algorithm. 
Then, the rest data is used to test the algorithm. It predicts around 80% of tweets correctly"
